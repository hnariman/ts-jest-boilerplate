# Boilperplate for ts + jest

Initial setup for project with TS & Jest

## Configuration steps

- git init
- yarn init
- tsc --init

- then install babel

```bash

yarn add -D babel @babel/preset-env @babel/preset-typescript

```

- update babel.config.js

- install jest & types:

```bash

yarn add -D jest @types/jest
```

- init jest config

```bash

jest init
```
